SEWA TEMPAT

Informasi lengkap untuk kebutuhan acara Anda

Berapa DP minimal?

DP minimal 50% untuk booking tanggal, sisanya dibayar H-7 acara. Pembayaran bisa via transfer bank atau cash.

Pembayaran
Boleh bawa vendor sendiri?

Boleh membawa vendor eksternal untuk dekorasi, fotografi, atau entertainment. Tetapi perlu koordinasi dengan event coordinator kami.

Vendor
Ada diskon weekday?

Ya! Diskon 10% untuk booking hari Senin-Kamis. Weekend (Jumat-Minggu) harga normal, dengan minimum booking 6 jam.

Diskon
Bisa extend waktu?

Bisa extend dengan biaya tambahan Rp 500.000 per jam. Harus konfirmasi maksimal 2 jam sebelum waktu berakhir.

Waktu
Bisa tour venue langsung?

Bisa! Silakan janji via WhatsApp dulu untuk jadwal tour. Buka untuk tour setiap hari 10:00-17:00 WIB.

Tour
Apa kebijakan refund?

DP tidak bisa refund, tapi bisa reschedule maksimal 1x. Pembatalan H-30 dapat refund 25%, H-14 tidak ada refund.

Kebijakan
Kapasitas maksimal?

Kapasitas maksimal 150 orang untuk standing party, 100 orang untuk seated dining. Fasilitas full AC dengan sound system profesional.

Kapasitas
Ada parkir?

Tersedia parkir untuk 30 mobil dan 50 motor. Gratis untuk tamu acara. Ada security untuk menjaga keamanan kendaraan.

Parkir
Ada sound system?

Sudah termasuk sound system profesional, mic wireless, dan lighting dasar. Bisa tambahan untuk kebutuhan spesifik dengan biaya tambahan.

Sound System
